## Основные понятия

### Алгоритм
**Алгоритм** — конечная последовательность действий, ведущая от исходных данных к требуемому результату.  
Свойства: дискретность, детерминированность, конечность, массовость, результативность.

### Блок-схема
Графическое представление алгоритма с помощью стандартных символов:

| Символ       | Название       | Назначение                             |
|--------------|----------------|----------------------------------------|
| ○            | Терминатор     | Начало / конец алгоритма               |
| □            | Процесс        | Выполнение действия                    |
| ◇            | Решение        | Проверка условия (ветвление)           |
| ▱            | Ввод/вывод     | Ввод данных или вывод результата       |
| →            | Линия потока   | Направление передачи управления        |

### Правила построения
- Чтение сверху вниз, слева направо.
- Каждый блок имеет один вход и (кроме условного) один выход.
- Ветви условия помечаются «Да» и «Нет».
- Избегать пересечений линий.
